How to Use

In order to use this plugin in your own application:

  1. import it as a dependency. If you use maven then do:

                                                    <groupId>com.github.orionlibs</groupId>
                                                    <artifactId>project-name</artifactId>
                                                    <version>0.0.1</version>
                                                </dependency>
